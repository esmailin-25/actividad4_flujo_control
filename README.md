## Descripción del Programa:

Este programa fue desarrollado utilizando pseudocódigo en PSeInt.

Permite ingresar una cantidad determinada de números enteros y luego determina la cantidad de números positivos, negativos, pares e impares.

## El sistema solicita:

- Cantidad de números a ingresar
- Los números uno por uno

Luego muestra los resultados indicando:

- Cantidad de números positivos
- Cantidad de números negativos
- Cantidad de números pares
- Cantidad de números impares

## Herramienta utilizada:

PSeInt
